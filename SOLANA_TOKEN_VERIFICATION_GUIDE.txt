💡 KEY CONTRACT FEATURES
═══════════════════════════════════════════════════════════════════════════════

✅ Admin Transfer-Free
   - Owner can transfer tokens without paying gas fee
   - Implemented in override transfer() function

✅ User Gas Fee Requirement
   - Regular users must call payGasFee() first
   - Users pay ETH to contract to get gas balance
   - Can only transfer if gasBalance > required fee

✅ Configurable Fees
   - Admin can change minGasFee and maxGasFee
   - setGasFeeBounds() function
   - Default: 0.1 ETH per transfer

✅ Approved Senders
   - Admin can whitelist addresses to bypass gas fee
   - setApprovedSender() function
   - Useful for partners, exchanges, etc.

✅ Custom Recipient Fees
   - Different fees for different recipients
   - setCustomGasFee() function
   - If set to 0, uses default minGasFee

✅ Gas Fee Withdrawal
   - Admin can withdraw collected ETH gas fees
   - withdrawGasFees() function
   - All collected ETH goes to admin wallet
